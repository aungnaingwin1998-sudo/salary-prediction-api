# 💼 Salary Prediction API

A production-style **Machine Learning REST API** built with **FastAPI**, **Scikit-learn**, and **Docker**.  
Given years of professional experience, the API returns a predicted annual salary in USD.

---

## 📁 Project Structure

```
salary-prediction-api/
├── app/
│   ├── __init__.py
│   ├── main.py          # FastAPI application & endpoints
│   ├── schemas.py       # Pydantic request/response models
│   └── model/
│       └── model.pkl    # Trained pipeline (generated by train.py)
├── train.py             # Data generation + model training script
├── requirements.txt
├── Dockerfile
├── .dockerignore
└── README.md
```

## 🤖 Model Details

| Item | Detail |
|---|---|
| Algorithm | `RandomForestRegressor` (200 trees) |
| Pipeline | `StandardScaler → RandomForestRegressor` |
| Training data | 1,000 synthetic samples |
| MAE | ~$3,450 |
| R² | ~0.96 |

## 🚀 Run Locally

### Prerequisites
- Python 3.10+

### 1. Clone & install dependencies

```bash
git clone <repo-url>
cd salary-prediction-api
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Train the model

```bash
python train.py
# ✅ Training complete
#    MAE : $3,450.35
#    R²  : 0.9609
#    Model saved → app/model/model.pkl
```

### 3. Start the API

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

The API is now live at **http://localhost:8000**  
Interactive docs: **http://localhost:8000/docs**

## 🐳 Run with Docker

```bash
# Build the image (trains the model automatically)
docker build -t salary-prediction-api .

# Run the container
docker run -d -p 8000:8000 --name salary-api salary-prediction-api

# Check logs
docker logs salary-api

# Stop & remove
docker stop salary-api && docker rm salary-api
```

## 📡 API Reference

### `GET /`
Returns API status.

```bash
curl http://localhost:8000/
```

```json
{
  "message": "Salary Prediction API is running.",
  "version": "1.0.0",
  "docs": "/docs"
}
```

---

### `GET /health`
Health check — confirms the ML model is loaded.

```bash
curl http://localhost:8000/health
```

```json
{
  "status": "ok",
  "model_loaded": true
}
```

Returns **503 Service Unavailable** if the model failed to load.

---

### `POST /predict`
Predict annual salary from years of experience.

**Request**
```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"years_experience": 3.5}'
```

**Response**
```json
{
  "predicted_salary": 38911.24,
  "years_experience": 3.5
}
```

**Validation rules**
| Field | Type | Min | Max |
|---|---|---|---|
| `years_experience` | float | 0 | 50 |

**Error example** (invalid input)
```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"years_experience": -1}'
```
```json
{
  "detail": [
    {
      "type": "greater_than_equal",
      "loc": ["body", "years_experience"],
      "msg": "Input should be greater than or equal to 0"
    }
  ]
}
```

## 🛠 Tech Stack

- **FastAPI** — async web framework with auto-generated OpenAPI docs  
- **Scikit-learn** — ML pipeline (preprocessing + regression)  
- **Pydantic v2** — request/response validation  
- **Joblib** — model serialisation  
- **NumPy** — numerical computation  
- **Docker** — multi-stage containerisation (slim runtime image)  
- **Uvicorn** — ASGI server
