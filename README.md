# 💼 Salary Prediction API

A production-style **Machine Learning REST API** that predicts annual salary from years of professional experience — built with **FastAPI**, **Scikit-learn**, and **Docker**, served with a clean web UI.

## ✨ Features

- 🤖 **ML pipeline** — StandardScaler + RandomForestRegressor
- ⚡ **FastAPI** — async REST API with auto-generated Swagger docs
- 🌐 **Web UI** — interactive salary calculator served from the API
- 🛡️ **Pydantic v2** — strict input/output validation
- 🐳 **Docker** — multi-stage build with health check
- 📊 **R² = 0.96** — high accuracy predictions

## 🚀 Run Locally

### Prerequisites
- Python 3.10+

### 1. Clone & install dependencies

```bash
git clone https://github.com/aungnaingwin1998-sudo/salary-prediction-api.git
cd salary-prediction-api
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Train the model

```bash
python train.py
# ✅ Training complete
#    MAE : $3,450.35
#    R²  : 0.9609
#    Model saved → app/model/model.pkl
```

### 3. Start the API

```bash
uvicorn app.main:app --reload --port 8000
```

| URL | Description |
|---|---|
| `http://localhost:8000` | Web UI |
| `http://localhost:8000/docs` | Swagger interactive docs |
| `http://localhost:8000/health` | Health check |

## 🐳 Run with Docker

```bash
# Build the image (trains the model automatically)
docker build -t salary-prediction-api .

# Run the container
docker run -d -p 8000:8000 --name salary-api salary-prediction-api

# Check logs
docker logs salary-api

# Stop & remove
docker stop salary-api && docker rm salary-api
```

## 📡 API Reference

### `GET /`
Serves the web UI (salary calculator).

---

### `GET /api`
Returns API status as JSON.

```bash
curl http://localhost:8000/api
```

```json
{
  "message": "Salary Prediction API is running.",
  "version": "1.0.0",
  "docs": "/docs"
}
```

---

### `GET /health`
Health check — confirms the ML model is loaded.

```bash
curl http://localhost:8000/health
```

```json
{
  "status": "ok",
  "model_loaded": true
}
```

Returns **503 Service Unavailable** if the model failed to load.

---

### `POST /predict`
Predict annual salary from years of experience.

**Request**
```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"years_experience": 3.5}'
```

**Response**
```json
{
  "predicted_salary": 38911.24,
  "years_experience": 3.5
}
```

**Validation rules**
| Field | Type | Min | Max |
|---|---|---|---|
| `years_experience` | float | 0 | 50 |
